# ✅ Advanced System Design – .NET Interviews

### ❓ Problem

Millions of users, flash sales, high read/write traffic.

### ✅ Architecture (Interview Expected)

    Client
      ↓
    API Gateway
      ↓
    Load Balancer
      ↓
    .NET 8 Web API (Minimal APIs)
      ↓
    │── Product Service

    │── Payment Service
