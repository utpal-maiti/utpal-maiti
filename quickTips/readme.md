# 🚀 MASTER .NET INTERVIEW PREP (ALL‑IN‑ONE)

## 🧩 INTERVIEW FLOW (What to Expect)

Most companies use **4–6 rounds**:

1.  Core C# + .NET
2.  Backend / API design
3.  Full‑Stack (UI + API integration)
4.  Azure / Cloud
5.  System design (Senior/Lead)
6.  Behavioral & Leadership

***

## 1️⃣ BACKEND DEVELOPER (Senior‑level)

### ✅ Must‑Know Topics

*   ASP.NET Core request pipeline
*   Middleware vs Filters
*   Minimal APIs vs Controllers
*   Authentication & Authorization
*   Performance optimization
*   Security best practices
*   Caching (memory + distributed)
*   Background services

***

### 🔥 Common Backend Questions

#### Q: How does ASP.NET Core handle a request?

✅ Expected senior answer:

*   Request enters middleware pipeline
*   Authentication → Authorization
*   Routing → Endpoint execution
*   Response flows back
*   Exception handling middleware catches errors

***

#### Q: How do you improve API performance?

✅ Say at least 6:

*   Async/await everywhere
*   `AsNoTracking()` for reads
*   Pagination
*   Caching (Redis)
*   Compression (gzip / brotli)
*   Connection pooling
*   Avoid over‑serialization
*   Use DTOs

***

#### Q: How do you secure APIs?

✅ Senior keywords:

*   JWT / OAuth2
*   Policy‑based authorization
*   HTTPS only
*   Token expiry & refresh tokens
*   Rate limiting
*   Input validation
*   Regular patching

***

## 2️⃣ FULL‑STACK (.NET + Angular / React)

### ✅ Architecture You Should Describe

    Angular / React
       ↓
    REST API (ASP.NET Core)
       ↓
    Application Layer
       ↓
    Domain Layer
       ↓
    EF Core / Dapper
       ↓
    SQL / NoSQL

***

### 🔥 Full‑Stack Questions

#### Q: How does Angular/React communicate with .NET API?

✅ Answer:

*   HttpClient / Axios / Fetch
*   JWT token in Authorization header
*   CORS configured in backend
*   API versioning

***

#### Q: How do you handle authentication in UI?

✅ Say:

*   Login → token stored (HttpOnly cookie preferred)
*   Token sent with each request
*   Refresh tokens
*   Role‑based UI rendering

***

#### Q: How do you deploy front‑end?

✅ Azure‑ready answer:

*   Angular/React build
*   Static Web Apps OR Blob Storage + CDN
*   Environment‑specific configs

***
